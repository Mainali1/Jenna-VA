## System Requirements

- Node.js >= 18.0.0
- npm >= 9.0.0

## Browser Compatibility

### Production
- Browsers with >0.2% market share
- Not dead browsers
- Not Opera Mini

### Development
- Latest Chrome version
- Latest Firefox version
- Latest Safari version
